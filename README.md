# Skinnovation LAB Website

**Modern multi-page website for Skinnovation LAB** — Premium aesthetic clinic in Brussels specializing in non-surgical fat freezing (Vetbevriezing).

## 🚀 Live Demo
[https://stijnman.github.io/skinnovationlab-remake](https://stijnman.github.io/skinnovationlab-remake)

## 📁 Project Structure

```
skinnovationlab-remake/
├── index.html          # Home page
├── over-ons.html       # About the clinic
├── behandelingen.html  # Treatments & Pricing
├── resultaten.html     # Before & After gallery
├── contact.html        # Contact + Google Maps
├── boek.html           # Booking page (with calendar)
├── .gitignore
└── README.md
```

## ✨ Features

- Fully responsive (mobile-first)
- Beautiful modern design
- Interactive booking calendar
- Google Maps integration
- Bilingual ready (NL/EN)
- Print-optimized
- Professional hamburger menu on mobile

## 🛠️ How to Run Locally

1. Download or clone this repository
2. Open any `.html` file in your browser
3. All pages are linked and fully functional

## 📤 Deployment

### GitHub Pages (Recommended)
1. Go to repository **Settings** → **Pages**
2. Source: Deploy from `main` branch
3. Your site will be live at: `https://stijnman.github.io/skinnovationlab-remake`

### Alternative Hosting
- Netlify
- Vercel
- Firebase Hosting
